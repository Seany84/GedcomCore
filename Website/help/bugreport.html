Reporting a Bug

If you found a bug, please file a entry in our bugtracker. Include the following information:

  • At which part of the library did you find the bug?
  • What happened?
  • What did you expect to happen?
  • Which version of Genea-GEDCOM do you use?
  • If possible, include code that reproduces the bug

We will try to solve the problem as soon as possible.
